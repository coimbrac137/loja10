# 🛒 LOJA 10 - Sistema de E-commerce Completo

## 📋 Sobre
Sistema de loja virtual com login, cadastro, produtos e carrinho de compras.

## ✅ Funcionalidades
- Login/Cadastro de usuários
- Catálogo de produtos (Descobertas do Dia + Eletrônicos)
- Busca em tempo real


## 🛠️ Tecnologias
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP 7.4+
- **Banco de Dados:** MySQL
- **Servidor:** XAMPP/WAMP/MAMP

## 📁 Estrutura do Projeto

**Páginas Principais**
- index.html → Login e Cadastro
- pagina2.html → Loja virtual

**Backend (api/)**
- config.php → Conexão com MySQL
- login.php → Autenticar usuário
- register.php → Cadastrar usuário
- logout.php → Encerrar sessão

**Banco de Dados**
- database.sql → Estrutura das tabelas

**Arquivos Estáticos (assets/)**
- banner.png → Imagem do banner
